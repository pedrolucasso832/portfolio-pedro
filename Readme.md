## 💻 Como Executar o Projeto

Clone o repositório:

```bash
git clone https://github.com/pedrolucasso832/portfolio-pedro.git
```

Acesse a pasta:

```bash
cd portfolio-pedro
```

Abra o arquivo `index.html` no navegador  
ou utilize a extensão Live Server no VS Code.
